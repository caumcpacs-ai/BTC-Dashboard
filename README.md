# Medical Data Dashboard

A professional medical data dashboard for visualizing and analyzing hospital data.

## Features
- Interactive charts (Hourly, Department, Weekday statistics)
- Automatic data import from Excel and Access (.accdb) files
- Real-time data synchronization
- Robust deduplication and data handling

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `npm start` (or `node server.js`)
4. Open `index.html` in your browser (or access via server if configured)

## Files
- `index.html`: Main dashboard interface
- `app.js`: Frontend logic and chart visualizations
- `server.js`: Backend server for data handling
- `style.css`: Dashboard styling
- `*.ps1`: PowerShell scripts for data processing and database interaction
